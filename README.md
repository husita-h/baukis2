## 推奨されるシステム環境

> - Ubuntu 18.04
> - Ruby 2.6.4
> - PostgreSQL 11.2

## 開発

### サーバー起動 (development 環境)

```
rails s -b 0.0.0.0
```

access to
http://example.com:3000/ or http://baukis2.example.com:3000/
